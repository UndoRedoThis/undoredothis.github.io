What is JSONScript?

JSONScript converts JSON objects on your webpage into valid Javascript-usable objects, functions, numbers, strings or whatever you like. Defining a parsable object is simple. Just create an object with the JSONScript key and you're good to go.

var myJsonScript = { "jsonscript": [] }

JSONScript will recognize those objects and start to parse them as soon as the page has been loaded successfully. For more information about the certain types supported by JSONScript, please consider paying the documentation a visit.

Working with JSONScript

Basic structure

Every thing you want JSONScript to parse requires one fundamental key-value-assignment in order to work: type. There are six basic types which originate from some of Javascript's design: object, function, number, string, array and call. Once you get the drill, you'll recognize that there are a lot of similarities between each one of them.

Object

var myJsonScript = { "jsonscript" : [ { "type" : "object", "name" : "myAwesomeObject" "children" : {[]} } ] }
